Q1: 

            function counter(){
                var counter = 0;
                
                function IncreaseCounter() {
                    return counter += 1;
                };
                
                return IncreaseCounter;
            }
                
            var counter = counter();
            alert(counter());
            alert(counter());
            alert(counter());
            alert(counter());

OUTPUT :- 4 time alert is shown with counter value 1,2,3,4 respectivly

NOTE:-

The value of counter will Increase with respect to how many times it is called, because the counter function, is getting updated as soon as it is called another time.

Q2 :- 

            let count = 0;
            (function () {
                if (count === 0) {
                    let count = 1;
                    console.log(count); // What is logged?
                }
                console.log(count); // What is logged?
            })();

OUTPUT :- 1
0

NOTE :-

when console.log(count) was count first time, there it matched the if condition, and when it was called second time it searched the value which was out of the function scope so it accessed the let variable of count i.e 0.

Q8 :- 

                var globalVar = "xyz";

                    (function innerFunc(innerArg) {
                        var innerVar = 'b';

                            "outerArg = " + outerArg + "\n" +
                            "innerArg = " + innerArg + "\n" +
                            "outerVar = " + outerVar + "\n" +
                            "innerVar = " + innerVar + "\n" +
                            "globalVar = " + globalVar);
                        })(456);
                    })(123);

OUTPUT :- outerArg = 123
innerArg = 456
outerVar = a
innerVar = b
globalVar = xyz

NOTE :-

Every variable and function is assigned with respected value
